# Light DNS Blocklist

This repository contains a filtered version of hagezi's DNS blocklist.

## Files
- **filtered.txt**: Contains only entries with IP 0.0.0.0
- **original.txt**: The original downloaded file

## Usage
Use this URL in Pi-hole, AdGuard, or your hosts file:
```
https://raw.githubusercontent.com/kevinleijh/lightadlist/main/filtered.txt
```

*Automatically updated daily*
